MQTT Collector

Subscribe to MQTT topics that Stats4Lox should directly push to the Influx database.

What is MQTT Collector good for?

With Stats4Lox's MQTT Collector, you can directly send data from MQTT devices to your Influx database, bypassing the way to the Miniserver and back. You need to subscribe the topics you want to collect. The MQTT Collector automatically expands json payload and true/false values. By default, only numeric values are collected. On a per-topic basis, you can enable value extraction from strings (e.g. get "12.4" value out of a "12.4Â°C" string), and also full string recording (Influx also supports strings as values).
MQTT Gateway is required, but subscriptions in MQTT Gateway (that are forwarded to the Miniserver) are independent of subscriptions in here.
